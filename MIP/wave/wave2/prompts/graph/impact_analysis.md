# Impact Analysis Graph

> Prompt 19 · Category: Graph · Skill: [graph-engineer](../../batch2/skills_framework/skills/graph-engineer/skill.md)
> Honors [MIP Engineering Principles](../../batch2/skills_framework/MIP_ENGINEERING_PRINCIPLES.md).

## Purpose
Determine impact of proposed changes.

## Context
Impact analysis is one of the highest-value modernization capabilities.

## Inputs
Knowledge Graph · AST Metadata · Control Flow Graph · Data Flow Graph · Target Node.

## Instructions
**Analyze:** Upstream Dependencies · Downstream Dependencies · Consumers · Producers · Business Capabilities · Applications · Interfaces · Data Flows · Transactions.

Generate impact report.
**Calculate:** Blast Radius · Risk Score · Confidence Score.

## Expected Output
- Impact Summary
- Affected Components
- Affected Capabilities
- Risk Assessment
- Recommended Actions
- Confidence Scores
- Blast Radius Analysis

## Constraints
Every affected component traced through evidence; every estimate carries a confidence score.

## Success Criteria
Engineers can estimate change risk before implementation.
