Key V2 change — resilience:
    The old absolutist constraint "Never assume. Never infer." has been replaced
    across all skills with a confidence-scoring + graceful-degradation model.
    Inference is allowed when LABELED with discovery method + confidence +
    validation status (Confirmed / Inferred / Needs Review) and flagged for
    review. The platform tolerates missing metadata, partial source, dynamic
    calls, and undocumented interfaces instead of failing — which is what makes
    the system resilient and able to understand the complete system.
