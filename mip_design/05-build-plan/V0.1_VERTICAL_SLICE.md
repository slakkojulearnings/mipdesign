# v0.1 — The Vertical Slice

> The single most important move in the whole project: stop designing, build one thin
> slice that *runs* and *measures itself*. This realizes Stage 1 (find the root/driver
> program) end-to-end and proves the spine so every later layer is incremental.

## Goal (verifiable)

Point MIP at a mainframe estate and answer, correctly and with evidence:
- **Which jobs execute program X?**  (root-driver query)
- **What does program X call / use / read / write?**
- **Which programs are roots (entry points) vs dead code?**

Success = `mip scan` + `mip query` produce the right answers on `sample_estate/`, and
`pytest` passes a ground-truth precision/recall check. Green run = done.

## Scope (and deliberate non-scope)

**In:** recursive scan + **content-based** classification (members have no extensions),
COBOL extraction (`PROGRAM-ID`, static `CALL`, `COPY`, `EXEC SQL` tables), JCL
extraction (`JOB`, `EXEC PGM=`), SQLite load against [`../01-metadata-model/schema.sql`](../01-metadata-model/schema.sql),
queries, a Typer CLI, and a ground-truth test.

**Out (documented, not hidden):** full COBOL grammar/AST, `COPY ... REPLACING`
expansion, CICS/IMS/MQ flow, capability clustering, LLM layer. These are later phases.

**Honest about the hard case:** dynamic `CALL WS-VAR` is detected and emitted as
`needs_review` / low confidence — never dropped, never `confirmed`.

## Build order (each step verifiable before the next)

1. **`schema.sql` + `models.py`** — already authored in [`../01-metadata-model`](../01-metadata-model). Reuse as-is.
2. **`scanner.py`** — walk a directory; classify each file by **content signature**
   (then folder, then extension as fallback). Emit `Artifact` rows. ✅ when counts match.
3. **`cobol.py`** — regex extractors → `Program`, `CALLS`/`USES`/`READS`/`WRITES`
   relationships; mark dynamic calls `needs_review`. ✅ when a known program's edges match.
4. **`jcl.py`** — extract `JOB` + `EXEC PGM=` → `Job`, `JobStep`, `EXECUTES`. ✅ when
   job→program rows match.
5. **`store.py`** — create the SQLite DB from `schema.sql`, insert entities + edges.
6. **`queries.py` + `cli.py`** — `mip scan PATH`, `mip query "..."`. ✅ when the
   root-driver question returns the right job(s).
7. **`tests/`** — a hand-labeled ground-truth file over `sample_estate/`; assert
   extracted calls/jobs and report **precision/recall**. ✅ when it passes.

## Apply the working rules ([`../CLAUDE.md`](../CLAUDE.md))
Minimum code that solves the problem; surgical; verified by tests, not by confidence.
The whole slice is a few hundred lines — keep it that way.

## After v0.1 (the path, not the promise)
Graph layer on NetworkX (root detection §1, blast radius §2 from
[`../02-algorithms/CORE_ALGORITHMS.md`](../02-algorithms/CORE_ALGORITHMS.md)) → real
COBOL grammar/AST → lineage → capability clustering → LLM Q&A. Each is additive on top
of a spine that already runs and is measured.
