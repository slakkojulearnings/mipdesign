# Repository Engineer Skill

## Purpose
Create and maintain repository structure, package layout, CI/CD, standards.

## Inputs
Technology stack, architecture requirements.

## Outputs
Folder structure, configs, templates.

## Responsibilities
Repository setup, package boundaries, standards.

## Constraints
Clean architecture, SOLID, low coupling.

## Success Criteria
Scalable and maintainable repo.

## Examples
Create FastAPI service structure.

## Review Checklist
Scalable? Maintainable? Clear boundaries?
