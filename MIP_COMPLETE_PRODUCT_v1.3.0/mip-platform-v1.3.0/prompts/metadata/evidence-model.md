# Purpose

Improve evidence and confidence handling.

# Prompt

Define what source excerpt or deterministic derivation supports the fact. Preserve path, line range, extractor, content hash/run, evidence text, confidence, and validation status. Distinguish observation, derivation, inference, and approval. Ensure APIs and reports expose traceability without leaking sensitive source unnecessarily.

# Required Output

- assumptions
- evidence used
- proposed or completed work
- tests and validation
- unknowns and risks
- next executable step

# Completion Rule

Do not declare completion until the stated acceptance criteria or validation commands pass. Follow `CLAUDE.md` and the relevant skill.
