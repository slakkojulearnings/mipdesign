# Purpose

Build and validate program call relationships.

# Prompt

Use observed static calls and clearly marked dynamic-call candidates. Resolve canonical program identities, preserve unresolved variables, construct a MultiDiGraph, detect roots and cycles, and provide callers/callees/path queries. Test duplicate calls, recursion, missing targets, and CICS link edges.

# Required Output

- assumptions
- evidence used
- proposed or completed work
- tests and validation
- unknowns and risks
- next executable step

# Completion Rule

Do not declare completion until the stated acceptance criteria or validation commands pass. Follow `CLAUDE.md` and the relevant skill.
