# Purpose

Assess whether an asset can be retired.

# Prompt

Check callers, executions, readers/writers, schedules, runtime evidence, rules, compliance retention, manual operations, reports, and unresolved references. Produce retain/retire/observe recommendation, confidence, required monitoring period, rollback, and approval criteria. Absence from static analysis is not proof of no use.

# Required Output

- assumptions
- evidence used
- proposed or completed work
- tests and validation
- unknowns and risks
- next executable step

# Completion Rule

Do not declare completion until the stated acceptance criteria or validation commands pass. Follow `CLAUDE.md` and the relevant skill.
