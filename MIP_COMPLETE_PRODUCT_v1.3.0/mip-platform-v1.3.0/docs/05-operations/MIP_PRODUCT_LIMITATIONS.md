# MIP v1 Product Limitations

MIP v1 is a complete runnable foundation, but enterprise mainframe semantics vary by compiler, site standards, preprocessors, scheduler, and data encoding.

Known extension areas:

- full COBOL grammar and compiler directives
- COPY REPLACING and nested replacement semantics
- full data-layout calculation for all USAGE/SYNC/variable OCCURS combinations
- complete JCL procedure expansion and symbolic substitution
- scheduler-specific calendars and event conditions
- full CICS, IMS, MQ, assembler, PL/I, VSAM, and utility semantics
- column-level SQL lineage
- runtime telemetry correlation
- production authentication/multi-tenancy
- distributed execution for very large estates

These are explicit roadmap items, not silently claimed capabilities.
