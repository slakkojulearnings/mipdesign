# MIP CLI and API Reference

The authoritative command help is generated by Typer:

```bash
mip --help
mip analyze --help
```

The authoritative REST contract is generated by FastAPI:

```text
GET /docs
GET /openapi.json
```

Core deterministic questions:

- callers and callees
- jobs executing a program
- root programs
- upstream/downstream lineage
- dependency impact
- latest statistics
- run comparison
- repository validation
