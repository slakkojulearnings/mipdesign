# MIP Observability

## Current v1 signals

- scan-run status and duration timestamps
- file/asset/relationship/issue counts
- per-file parse status
- persisted parser issues
- deterministic validation report
- generated HTML and JSON report

## Enterprise extensions

Instrument pipeline phases with OpenTelemetry traces, counters, histograms, and structured logs. Track parser latency, classification distribution, unresolved-reference rate, metadata coverage, graph-build time, query latency, and failure/retry counts. Never log source contents or sensitive values by default.
