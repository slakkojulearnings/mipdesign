Phase 2: Understand Mainframe Building Blocks

1. JCL 
My thoughts are :
- JCL means : 
    Windows .bat file 
    Linux shell script
- JCL answers:
    What runs?
    When?
    With what files?

ex: //STEP01 EXEC PGM=ABC100 
meaning : Run program ABC100

2. COBOL

My thoughts are : Business rules

ex: 
IF BALANCE > LIMIT
    MOVE 'Y' TO DECLINE
END-IF

Translation: Decline transaction if balance exceeds limit.

3. Copybooks
 
My thoughts are : 
    Header files
    Shared data structures
ex: 
COPY CARDREC.

Contains :
    Card Number
    Customer Number
    Balance
    Status

4. DB2

My thoughts : SQL database

Stores:
    Card Number
    Customer Number
    Balance
    Status

5. CICS

My thoughts - Web application server

before web apps existed :
    Customer Service Screen
    Authorization Screen
    Maintenance Screen

Phase 3: Find Entry Points - They open random COBOL programs.

- Find all CICS Transations - Search transaction definitions.

Now, you will know where online starts
